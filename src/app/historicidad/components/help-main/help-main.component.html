Dashboard Registros Historicos

Descripción

En esta seccion se busca entregar la funcionalidad de explorar y generar visualiazciones genericas de los datos historicos registrados por cada sensor de cada estacion. Las visulizaicon generadas dependerar de los parametros configurables, los cuales abordan tres puntos:

  • Estaciones: Seleccion de una o varias estacion con un maximo de 5. Segun las estaciones seleccionadas, las variables disponibles cambiaran.

  • Variable (sensor): Lista de variables presentes en al menos una estacon de las estaciones seleccionadas. Las variables hacen refencia a los sensores disponibles con su altura respecto al suelo de la estacion. Cabe señarlar que la lista de variables disponible se genera relizando una union entre la lista de sensores de cada estacion seleccionada, por lo que es altamente probable de que no todas las estacion cuente con alguna variable.

  • Procesamiento Temporal: este punto hace refencia a los parametros de rango temporal, agrupacion temporal y tipo de operacion agregada a plicar sobre los grupo. Habiendo escogido la fuente de datos (estacion + variable), se tiene que indicar el perido de analisis de interes y el como se procesaran los datos esta ventana temporal

Los parametros deben ser configurados en este orden.

Seleccion de Estaciones (Mapa)

Para seleccionar las estaciones de interes hay dos formas, (1) mediante la seleccion directa (clickeando) en el mapa o (2) mediante la barra de busqueda ingresando el nombre de la estacion. toodas las estaciones que se selccionen quedan registradas en la barra superior.

Seleccion de multiples estaciones

Es importante resaltar que hay dos modos de seleccion, seccion unica y seleccion multiple y esta se escoje con el checkbox ubicado e la parte superior de la barra lateril de configuracion de paramentros.

Tenga en cuenta que dependiendo del modo seleccion escogico (multiple o unica), puede redirigir distintos dashboards secciones.

Barra Lateral de Opciones (Variable y Procesamiento Temporal)

Sección del dashboard que se encuentra al lado izquierdo de la pantalla. Este barra lateral permite seleccionar la variable de interes y las opciones de procesamiento temporal. Importante!!!, esta seccion no se desbloqueara hasta que se haya seleccionado almenos una estacion.

  • (1) Variable: Este combobox permite seleccionar una variable presente en las en las estaciones. todo lo relativo a este campo se explico en al seccion de descripcion.

  • (2) Parametros de procesamiento temporal

    Como ya se menciono en la seccion de descripccion, en estas 3 casillas se deben escoger el rango de timpo de interes, la forma que se agruparan los datos dentro de rango y que operacion agregada se aplicara sobre ellos. Las formas de agrupaacion temporal son: diaria (00:00Hr a 23:59Hr), semanal (lunes 00:00Hr a domingo 23:59Hr), mensual.
